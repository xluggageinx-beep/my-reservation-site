# 선문대학교 치위생학과 - 대상자 실습 지원 시스템

선문대학교 치위생학과의 실습 대상자 예약 관리를 위한 웹 애플리케이션입니다.

**Developed by 치위생학과 23학번 이지민**

## 프로젝트 개요

이 시스템은 치위생학과 학생(술자)과 실습 대상자를 연결하여 효율적인 실습 일정 관리를 지원합니다. 대상자는 원하는 시간대(타임)와 술자를 선택하여 예약할 수 있으며, 관리자는 3단계 권한 시스템(술자/RS/전체관리자)을 통해 예약을 관리할 수 있습니다.

## 주요 기능

### 완료된 기능

#### 1. 대상자 모드
- ✅ **메인 페이지**: 실습 프로그램 소개 및 안내 
  - 참여 혜택 (전문가 치면세마, 정밀 구강 검사, 맞춤형 교육, 퍼스널 케어)
  - 관리 과정 및 소요 시간 (1차/2차/최종회차)
  - 안전한 실습 환경 (전문적인 지도, 개인정보 보호, 철저한 감염 관리)
  - 유의사항 (무료 프로그램 안내, 참가비 관련 안내 포함)
- ✅ **개인정보 동의**: 이름, 전화번호, 구강건강관리 프로그램 동의서
- ✅ **개인정보 입력**: 대상자 이름, 전화번호, 술자와의 관계 입력
  - 관계 선택지: 직접입력, 가족, 치위생학과 선후배, 친구, 동아리, 물리치료학과
- ✅ **타임별 술자 선택**: 
  - 한 페이지에서 모든 타임과 술자 확인 가능
  - 타임 정렬: 월 → 화 → 수 → 목 → 금 순서, 같은 요일 내에서 이른 시간 순
  - 술자 정렬: 먼저 추가된 술자가 앞 순서
  - 스크롤로 전체 리스트 확인
- ✅ **예약 날짜 선택**: 선택한 술자의 주차별 예약 가능 날짜 선택
- ✅ **예약 확인 및 완료**: 예약 정보 확인, 최종 예약 완료, 캡처 안내 및 참고사항 포함

#### 2. 술자 모드 (관리자)
- ✅ **패스워드 인증**: 패스워드 '0814'로 관리자 인증
- ✅ **타임 수정**:
  - 타임 추가/수정/삭제 (최소 2개, 최대 6개)
  - 요일 및 시간 범위 설정
  - 주차 정보 수정 (연도별 캘린더에서 13개 날짜 선택)
  - 로딩 오류 방지 및 재시도 기능 추가
- ✅ **술자 리스트 수정**:
  - 술자 추가/수정/삭제 (한 타임당 최대 12명)
  - 이름, 학번, 전화번호, 소속 타임 관리
  - 술자 삭제 시 예약 데이터도 함께 삭제
- ✅ **예약 관리**:
  - 전체 예약 목록 조회 및 통계
  - 타임별 예약 현황 표시
  - 전체 예약 일괄 삭제 기능 (이중 확인)
- ✅ **예약 정보 확인** (3단계 권한):
  - **일반 술자**: 이름과 학번으로 본인 인증, 본인의 예약 목록 조회 및 취소
  - **RS(타임 담당자)**: 타임 이름 (예: 월요일 1T) + 패스워드 '0000' 입력으로 해당 타임 전체 술자의 예약 조회 (제한된 정보)
  - **전체 관리자** (히든 기능): "관리자" + 패스워드 '0000' 입력으로 전체 타임·전체 술자의 모든 예약 조회 및 관리 (표 형식, 전체 정보 표시)

#### 3. 시스템 기능
- ✅ **실시간 데이터 동기화**: RESTful API를 통한 데이터 관리
- ✅ **예약 중복 방지**: 
  - 한 대상자는 한 술자에게만 예약 가능
  - 한 술자의 특정 날짜는 한 명만 예약 가능
- ✅ **날짜 필터링**: 
  - 지난 날짜는 선택 불가
  - 이미 예약된 날짜는 선택 불가
- ✅ **반응형 디자인**: PC와 모바일 모두 지원, 모바일 폰트 최적화
- ✅ **전문적인 UI**: 흰 바탕(#ECECF0)에 파란색(#0059D7) 포인트
- ✅ **사용자 안내**: 무료 프로그램 안내, 참가비 관련 안내 포함

## 페이지 구조 및 URI

### 대상자 모드 페이지
- `/index.html` - 메인 페이지 (프로그램 소개)
- `/consent.html` - 개인정보 활용 동의
- `/participant.html` - 개인정보 입력
- `/selection.html` - 타임별 술자 선택
- `/reservation.html?timeId={id}&operatorId={id}` - 예약 날짜 선택
- `/success.html` - 예약 완료

### 술자 모드 페이지
- `/operator.html` - 관리자 페이지 (패스워드 인증 필요)
- `/reservation-check.html` - 예약 정보 확인 페이지

## 데이터 모델

### 1. Times (타임)
```javascript
{
  id: string,              // 고유 ID
  name: string,            // 타임 이름 (예: "1T", "2T")
  day_of_week: string,     // 요일 (월, 화, 수, 목, 금)
  time_range: string,      // 시간 범위 (예: "09:00-12:00")
  selected_dates: array    // 선택된 13개의 날짜 (YYYY-MM-DD)
}
```

### 2. Operators (술자)
```javascript
{
  id: string,              // 고유 ID
  name: string,            // 술자 이름
  student_id: string,      // 학번
  phone: string,           // 전화번호
  time_id: string          // 소속 타임 ID
}
```

### 3. Reservations (예약)
```javascript
{
  id: string,                  // 고유 ID
  participant_name: string,     // 대상자 이름
  participant_phone: string,    // 대상자 전화번호
  operator_id: string,          // 술자 ID
  time_id: string,              // 타임 ID
  reservation_date: string,     // 예약 날짜 (YYYY-MM-DD)
  consent_agreed: boolean       // 동의서 동의 여부
}
```

### 술자 모드 접속
1. 메인 페이지에서 "술자모드" 버튼 클릭
2. 패스워드 입력: `0814`
3. **타임 수정**, **술자 리스트 수정**, **예약 정보 확인** 가능

## 사용 흐름

### 대상자 예약 과정
1. 메인 페이지에서 "대상자 지원하기" 클릭
2. 개인정보 활용 동의 (이름, 전화번호, 프로그램 동의서)
3. 개인정보 입력 (이름, 전화번호)
4. 타임별 술자 리스트에서 원하는 술자 선택
5. 예약 가능한 날짜 중 선택
6. 예약 정보 확인 및 최종 확정
7. 예약 완료 및 주의사항 안내

### 관리자 설정 과정
1. 메인 페이지에서 "술자모드" 클릭
2. 패스워드 입력 (0814)
3. **타임 수정**:
   - 타임 추가/수정/삭제 (2~6개)
   - 요일 및 시간 설정
   - 주차 정보 수정 (연도별 캘린더에서 요일에 맞는 날짜 13개 선택)
4. **술자 리스트 수정**:
   - 술자 추가/수정/삭제 (한 타임당 최대 12명)
   - 이름, 학번, 전화번호, 소속 타임 입력
5. **예약 정보 확인**:
   - 이름과 학번으로 본인 인증
   - 본인의 예약 목록 조회
   - 필요 시 예약 취소 (지난 예약 제외)

## 주요 제약사항

- ✅ 타임은 최소 2개, 최대 6개까지 생성 가능
- ✅ 한 타임당 최대 12명의 술자 추가 가능
- ✅ 한 타임당 13개의 날짜 선택 가능
- ✅ 한 대상자는 한 술자에게만 예약 가능
- ✅ 한 술자의 특정 날짜는 한 명만 예약 가능
- ✅ 한 술자에게 1~3회 내원 가능
- ✅ 지난 날짜는 예약 불가
- ✅ 부득이한 취소 시 최소 3일 전 사전 연락 필요

## 주의사항

### 대상자 안내사항
- 실습에 참여 전 예약 받은 실습생이 개별 연락 예정
- 연락이 오지 않으면 담당자 연락처로 문의
- 교육 과정 특성상 예약 시간 준수 매우 중요
- 부득이한 취소 시 최소 3일 전 사전 연락 필수
- 여러 실습생에게 예약 불가 (한 실습생에게 1~3회 내원 가능)

### 관리자 안내사항
- 술자 모드 패스워드: `0814`
- 타임 삭제 시 소속 술자가 있으면 삭제 불가
- 술자 삭제 시 관련 예약도 함께 삭제됨
- 주차 정보 수정 시 선택한 요일과 같은 요일만 선택 가능

## 문제 해결

### 자주 발생하는 문제

**Q: 술자 선택 페이지가 로딩되지 않아요.**
A: 다음을 확인해주세요:
- 개인정보 동의 및 참가자 정보를 먼저 입력했는지 확인
- 관리자가 타임과 술자를 먼저 등록했는지 확인
- 브라우저 콘솔(F12)에서 오류 메시지 확인
- 페이지를 새로고침 해보세요
**Q: 패스워드를 잊어버렸어요.**
A: 코드에서 `OPERATOR_PASSWORD` 변수를 확인하세요. 현재는 '0814'입니다.

**Q: 예약이 안 돼요.**
A: 다음을 확인해주세요:
- 모든 동의 항목에 체크했는지
- 개인정보를 올바르게 입력했는지
- 선택한 날짜가 과거가 아닌지
- 해당 날짜가 이미 예약되지 않았는지

**Q: 타임을 삭제할 수 없어요.**
A: 해당 타임에 속한 술자가 있으면 삭제할 수 없습니다. 먼저 술자를 삭제하거나 다른 타임으로 이동시켜주세요.
